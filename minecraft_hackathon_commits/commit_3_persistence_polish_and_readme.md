=================================================
1. ДОБАВИТЬ СОХРАНЕНИЕ МИРА В localStorage
=================================================

В script.js написать:
- const WORLD_STORAGE_KEY = "minicraft_world";

Добавить функцию saveWorld():
- localStorage.setItem(WORLD_STORAGE_KEY, JSON.stringify(world));

Добавить функцию loadWorld():
- прочитать localStorage.getItem(WORLD_STORAGE_KEY)
- если данных нет, вернуть false
- если данные есть:
  - попытаться JSON.parse(...)
  - записать в world
  - вернуть true
- если parse упал:
  - вернуть false

Лучше сделать через try/catch только внутри loadWorld().

Важно:
- не оборачивать в try/catch весь проект
- не молча глотать ошибки вне загрузки localStorage

2. ИЗМЕНИТЬ СТАРТОВУЮ ИНИЦИАЛИЗАЦИЮ МИРА
