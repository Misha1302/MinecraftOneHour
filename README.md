# MiniCraft

Minecraft-like game built with plain HTML, CSS, and JavaScript.

It renders a compact isometric block world without third-party libraries, then makes that world get weird on purpose.

## Features

- Generated voxel terrain with a larger explorable island
- Block breaking, placing, and a hotbar
- Chaos meter for the world's immediate anger
- Collapse meter for long-term run danger
- Three hidden anomaly cores to find and collect
- Parasite and flesh growth that can spread out of control
- Directional core hints during exploration
- Relief windows after stabilizing cores or burning parasites
- Win state for collecting every anomaly core
- Lose states for world collapse or parasite pressure
- Visual madness events, fake debug corruption, screen drift, and reality blinks
- World persistence in localStorage
- Clean new run reset by key R
- Meme world phrases on key F
- Fake debug overlay on key `

## Controls

- WASD: move
- Q/E: up/down
- LMB: break
- RMB: place
- 1-6: select block
- F: show random world phrase
- `: toggle fake debug overlay
- R: reset world

## How to run

Open `index.html` in a browser.

## Demo note

Find anomaly cores, keep parasite pressure down, and avoid letting world anger stay high. Press R after a win or loss to immediately start a clean run.
