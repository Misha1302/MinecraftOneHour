# MiniCraft

Minecraft-like game built with plain HTML, CSS, and JavaScript.

It renders an isometric block world without third-party libraries.

## Features

- Generated voxel terrain
- Block breaking and placing
- Hotbar
- World persistence in localStorage
- Reset by key R
- Meme world phrases on key F
- Fake debug overlay on key `

## Controls

- WASD: move
- Q/E: up/down
- LMB: break
- RMB: place
- 1-6: select block
- F: show random world phrase
- `: toggle fake debug overlay
- R: reset world

## How to run

Open `index.html` in a browser.

## Demo note

Press R when you need to show a clean world.
